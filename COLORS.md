# Theme Colors

| Variable | Usage | Default |
|----------|-------|---------|
| `bgMain` | Основной фон окна | `#1A1A2E` |
| `bgPanel` | Фон панелей (левая, правая), фон иконок | `#13294B` |
| `bgButton` | Фон кнопок Load / Edit | `#1A2A4A` |
| `bgButtonHover` | Фон кнопки Settings при наведении | `#221A2A4A` |
| `bgDialog` | Фон диалога сохранения | `#2A2A4A` |
| `bgOverlay` | Затемнение (пауза, диалог сохранения) | `#AA000000` |
| `textPrimary` | Основной текст на панелях, надписи Lines/Size | `#E0EDFF` |
| `textHint` | Подсказки в пустых полях | `grey` |
| `textOnButton` | Текст активных кнопок (нажатое состояние) | `white` |
| `textActiveLine` | Текущая строка лирики (первая из visible) | `#F0C040` |
| `textPause` | Текст паузы, предстроки, preview в countdown | `#FFCC00` |
| `textError` | Текст ошибки (невалидный JSON и т.д.) | `#CCFF0000` |
| `textStatusBar` | Строка состояния внизу окна | `white 30%` |
| `accentBorder` | Рамка диалога сохранения | `#4444AA` |
| `accentClose` | Крестик закрытия диалога | `#6666CC` |
| `sliderTrack` | Трек слайдера Lines / Size | `#4A6FA5` |
| `sliderThumb` | Ползунок слайдера Lines / Size | `#A0C4FF` |
| `countdown` | Цифры обратного отсчёта 3, 2, 1, GO! | `#44FF44` |
| `iconPrimary` | Обычный цвет иконок (text-align-left, squares-four, gear) | `#E0EDFF` |
| `iconHover` | Шестерёнка при наведении | `#F0C040` |
| `editorBg` | Фон редактора текста | `#13294B` |
| `editorText` | Текст в редакторе | `#E0EDFF` |
| `editorCaret` | Курсор в редакторе | `#E0EDFF` |
| `editorHighlight` | Выделение в редакторе | `#1A2A4A` |
| `editorHighlightedText` | Текст выделения | `white` |
| `editorOutline` | Рамка редактора | `transparent` |
